## 既知の運用課題(再ハマり防止)

### 課題1: AWS CLI for Windows の `file://` UTF-8 デコードエラー

**症状**:

```
aws cloudformation validate-template --template-body file://infrastructure/template.yaml ...
→ Error: Unable to load paramfile, text contents could not be decoded
```

日本語コメントを含む YAML が Windows 版 aws cli 2.12.6 で `file://` 経由で読み込めない。

**回避策(動作確認済み)**:

```powershell
$body = Get-Content infrastructure/template.yaml -Raw -Encoding UTF8
aws cloudformation validate-template --template-body $body --profile AWS-security-check --region ap-northeast-1
```

**恒久対策候補**: CFn テンプレートのコメントを英語に統一する(Phase 1.2 で着手時に検討推奨)。

### 課題2: `uv run cfn-lint` が `program not found`

**症状**:

```
uv run cfn-lint ../infrastructure/template.yaml
→ error: Failed to spawn: cfn-lint, Caused by: program not found
```

`.venv\Scripts\cfn-lint.exe` 自体は存在する(47104 bytes)が、`uv run` 経由で発見されない。Windows + uv 0.9.30 の挙動と推察。

**回避策(動作確認済み)**:

```powershell
.\.venv\Scripts\cfn-lint.exe ..\infrastructure\template.yaml
```

cfn-lint は `python -m cfnlint` も使えない(`__main__` がないパッケージ)。

### 課題3: Kiro ターミナル UI の表示乱れ

長いコマンドや日本語パスでカラム破綻が発生する。コマンド自体は正常実行されているので、出力末尾の結果を読み取れば判断できる。新セッションでも同じ現象を見たら気にしないこと。

### 新たに発覚した既知の運用課題

#### 課題4: `execute_pwsh` の `cwd` 指定が反映されない

**症状**: tool に `cwd: c:\...\kiro` を指定しても、シェルが `backend` ディレクトリに居続ける。

**回避策(動作確認済み)**: コマンド内のファイルパスを **絶対パス**で指定する。

```powershell
# NG
$body = Get-Content -Raw -Encoding UTF8 infrastructure\template.yaml; ...
# OK
$body = Get-Content -Raw -Encoding UTF8 'C:\_oka\a_資格関係\1_AWS\1_AS部\kiro\infrastructure\template.yaml'; ...
```

#### 課題5: `execute_pwsh` で `powershell -Command "..."` ラッパーが変数展開を壊す

**症状**: シェルが PowerShell 7 で動作しているため、`powershell -NoProfile -Command "$body = ..."` のように外側ラッパーで包むと、外側 pwsh が `$body` を空文字に展開してから子プロセスへ渡してしまい、コマンドが破綻する。

**回避策(動作確認済み)**: `powershell -Command "..."` ラッパーを **使わず**、直接コマンドを発行する(シェル自体が既に pwsh)。

#### 課題6: ターミナル出力に過去コマンドの残骸が混入

**症状**: 出力冒頭に `$TAG_NAME_BE = ...` や `Copy-Item ...` など、無関係なコマンド残骸が表示される。**コマンドの実行自体は成功**しているが見た目が乱れる。

**回避策**: 末尾の本来出力(JSON や `EXITCODE: N`)で判断する。これは課題3(ターミナル UI の表示乱れ)の延長現象。
